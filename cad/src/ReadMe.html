Nanorex NanoEngineer-1 Version 1.0.1

NanoEngineer-1 (NE1) is an open-source (GPL) 3D multi-scale modeling and simulation program for nano-composites with special support for structural DNA nanotechnology. It features an easy-to-use interactive 3D graphical user interface for designing and modeling large, atomically precise composite nanosystems.

This is NanoEngineer-1 Version 1.0.1 as of May 6, 2008.

System Requirements

Minimum

  • 2.5 GHz Intel Pentium 4 CPU or equivalent
  • Windows XP, Mac OS X 10.3.9, Linux
  • 1 Gb RAM
  • 100 Mb available disk space
  • OpenGL compatible 3D video card

Recommended

  • 3 GHz Intel Core 2 CPU or equivalent
  • Windows XP, Mac OS X 10.3.9, Linux
  • 2 Gb RAM
  • 100 Mb available disk space
  • OpenGL compatible 3D video card with FBO and VBO support (this web page gives a good list of recommended video cards.)

Installation Instructions

Windows

Run the NanoEngineer-1 self-extracting installer "NanoEngineer-1_1.0.1.exe".

Mac OS X

Run the NanoEngineer-1 self-extracting installer "NanoEngineer-1_1.0.1.dmg".

Note: If during installation, the message: "You cannot continue. There is nothing to install." appears, navigate to <your hard drive>/Library/Receipts/ and delete any files that start with "NanoEngineer-1". Then empty the Trash and run the installer again.

Extensions for NE1

NE1 makes use of a number of free extensions and companion applications, including:
  • GROMACS+HDF5, an extremely high performance, open source, and cross-platform package used to perform molecular dynamics simulation of systems with hundreds to millions of particles. The +HDF5 comes from our experimental HDF5 file format augmentation. NE1 uses GROMACS+HDF5 as an energy minimization engine.
  • Open Babel, used to import/export different chemistry file formats to/from NE1.
  • QuteMolX, a high-quality molecular visualization system that can be "launched and loaded" from NE1. Very cool!
  • POV-Ray, which creates high-quality renderings of the current model with a single click of the mouse.
  • MegaPOV (for Mac OS X), which creates high-quality renderings of the current model with a single click of the mouse. POV-Ray is also required.
  • GNUPlot, used to plot the results of molecular dynamics simulations using NanoDynamics-1, NE1's own molecular dynamics engine for nanomechanical simulations. It is installed with NE1 for Windows and Mac.
For more details on all extensions and links to download pages, visit: NanoEngineer-1 Extensions

Note that the NanoEngineer-1 Application Suite includes all of the above extensions and companion applications in one convenient installer. You can find it at www.nanorex.com.

New Features Archives

What is new in NanoEngineer-1 1.0.0 (4/25/2008):
  1. Core User Interface Operations Support
    • A new DNA data model has been implemented that makes use of a unique DNA structure checking mechanism.
  2. User Interface Improvements and Additions
    • DNA Strand Operations
      • New handles for resizing segments and strands along the axis.
      • Nick/break/join strands - now you can break a strand or join two strands via a break/join command on the toolbar.
      • Create a crossover in one simple operation by selecting nucleobases from adjacent strands and invoking the context menu.
    • DNA Segment Operations
      • Change the length of segment
        • Constrained to central axis
        • By adding base pairs
      • Rotate a segment about its central axis
      • Translate a segment along its central axis
    • DNA Duplex Operations
      • Create a single DNA duplex by selecting two points on a plane
        • The duplex is shown while you're selecting the second point and can be displayed in either ladder or ribbon display styles.
        • The duplex adheres to intelligent horizontal and vertical "snap-to-grid" rules allowing for rapid, horizontal and vertical placement.
        • Text next to the mouse cursor dynamically updates the number of base-pairs and the length of the duplex as you change the rubberband line during DNA duplex creation.
    • Sequence Editor
      • Load sequences from a text file, paste them in, or even type them.
      • Unique find/replace feature lets you quickly find and replace sub-sequences.
      • Easily switch between 5'-3' and 3'-5' orientation.
      • Save sequences to a separate text file.
    • Peptide Generator
      • This experimental module can create atomistic models of polypeptide chains of arbitrary sequence and secondary structure.
      • The polypeptide chain can be built step-by-step by choosing consecutive amino acids from a set of the standard 20 naturally occurring amino acids.
      • Local geometry of the built chain can be chosen from a set of predefined secondary structure geometries (alpha helix, 3-10-helix, pi-helix, or beta-strand) or explicitly defined by specifying a pair of phi/psi angles.
      • A sequence of the peptide is visualized in a sequence viewer window and it is colored by current secondary structure type.
      • Note:
        • Generated structures can't be minimized or simulated in a reliable way because the ND1 force field currently lacks certain atom types (e.g. amide-type nitrogen). The generated geometry is idealized, but after minimization it becomes wrong.
        • The structure is not visualized as it is being built, you need to press the "Preview" button to see it.
        • The structure can't be modified after it has been generated, nor can the sequence be edited.
    • Miscellaneous Improvements and Additions
      • DNA generator now supports PAM3 and PAM5 models, but not atomistic models.
      • A new rainbow-effect cylinder display style is now available for viewing reduced model PAM DNA structures.
      • A new nanotube generator - create a single-walled carbon nanotube by selecting two points on a plane.
      • Toolbar and dockwidget positions can now be saved between sessions.
      • Option to maintain the aspect ratio while resizing a plane. The ratio is updated automatically (i.e. anytime the plane's width and/or height are changed).
      • The bond offset sphere checkbox in Extrude property manager is back.
      • Mirror feature enhancements - it can now mirror everything but the planes.
      • Triple-click any PAM atom to select all connected PAM atoms.
      • New paste mode and related enhancements
        • New paste mode which can be accessed using menu Edit > Paste from clipboard... or corresponding button in the standard toolbar. You can return to the previous mode by hitting hitting the 'Done' or 'Escape' key.
        • Clipboard groups are now 'pastable' (so those are also included in the clipboard groupbox of Paste Mode)
        • Edit > Paste or Ctrl + V does a 'single-shot' paste operation. It pastes the most recently copied item. The item placement position is slightly modified compared to the original copied item.
        • Improvements to group pasting via paste mode. Now the groups are pasted at a closer place to where you double-clicked.
      • New part library mode - the part library can be accessed in any mode by using Insert > Part from partlib... menu. You can return to the previous mode by hitting the 'Done' or 'escape' key.
      • When a single atom is selected, a new groupbox in the Build Atoms Property Manager, Selection Options, becomes active and shows the x, y, z coords of the selected atom. You can then use the spinboxes to move the atom precisely.
      • NE1 can now be shown in full-screen mode (only menus and rulers appear around the work area) and semi-full-screen mode (includes all the toolbars as well.)
  3. Structural DNA Nanotech (SDN) Part Library Additions
    • Several DNA double-crossovers such as DAE, DAO, DPE, DPON, DPOW, DX Tile, DX+J Tile, JX2, and PX65
    • Several SDN structures such as Mao's 3- and 6-point stars, Rothemund's star origami, Pierce's walker, Turberfield's tetrahedron, and Shih's octahedron.
  4. PAM5 and PAM3 Reduced DNA Models
    • The original PAM5 has undergone further refinements to improve the realism of minimized SDN structures
    • A new PAM3 model has been added for simplified creation, and clear, attractive display of SDN motifs
  5. QuteMolX Integration
    • QuteMolX is our branch of QuteMol. The branch allows us to continue to improve the code, and better integrate it with NE1.
    • QuteMolX now matches the display mode, background color, and atom coloring of NE1 when either launched from NE1, or when opening PDB files exported from NE1.
  6. GROMACS Integration
    • GROMACS has been added as an energy minimization engine. Structures can be minimized in the foreground, or in a separate background GROMACS process. The location of all the GROMACS input/output files, and the identifier (PID) for the GROMACS process is printed in the History text area at the bottom of NE1. An application called NanoVision-1, which is currently in development and not yet available, will be able to attach to, monitor, and cleanly abort the spawned GROMACS processes.
    • HDF5 support was added to GMX for writing and viewing trajectory files, but won't be fully realized until NanoVision-1 is available.
  7. Notable Bug Fixes
    • The NE1 process now reliably ends upon exit of NE1.
    • POV-Ray scene fixed - bonds are now the correct color when a chunk is displayed in ball-and-stick style, and colored.
    • PAM reduced DNA models are now rendered properly in QuteMolX.
    • Fixed the "Rotate as a unit" checkbox in the Rotate groupbox.
    • Moiety copies are now placed closer together when extruding to a ring as final product.
    • Inserting a part with a hotspot from the parts library now works.
    • Newly read mmp files are no longer immediately marked as changed.
    • Added a mouse wheel governor for Mac users - using the mouse wheel to zoom is now manageable.
