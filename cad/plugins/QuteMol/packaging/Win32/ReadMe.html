QutemolX

This the Nanorex branch of Qutemol which has been modified to be a plug-in extension to NanoEngineer-1.

Installation notes:

If you have not installed this from the main NanoEngineer-1 installer you will have to modify your preferences to allow NanoEngineer-1 to use this plugin.

- To do this, go to Tools -> Preferences -> Plug-ins

- Check the box for QutemolX and input the full path (including file name) to the QutemolX.exe file which this package installed.

(ex. c:\Program Files\QutemolX\QutemolX.exe)
