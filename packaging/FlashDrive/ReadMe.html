Welcome to the Nanorex NanoEngineer-1 1.0.0 Flash Drive

This flash drive includes the entire NanoEngineer-1 suite of applications, plus bonus material.

Here's what's included:

  • Mac and Windows installers plus all source code for:
    • NanoEngineer-1 Suite - a single installer for the following applications
    • NanoEngineer-1 - an open-source (GPL) 3D multi-scale modeling and simulation program for nano-composites with special support for structural DNA nanotechnology. It features an easy-to-use interactive 3D graphical user interface for designing and modeling large, atomically precise composite nanosystems. Learn more...
    • GROMACS+HDF5 - an extremely high performance, open source, and cross-platform package used to perform molecular dynamics simulation of systems with hundreds to millions of particles. NE1 currently uses GROMACS+HDF5 as an energy minimization engine option. Learn more...
    • QuteMolX - an interactive, high quality molecular visualization application aimed at improving clarity and understanding of the 3D shape and structure of large complex molecules. Users can "launch-and-load" QuteMolX directly from NE1. Learn more...
  • The Productive Nanosystems nanofactory film. A computer-generated animated short film depicting an animated view of a nanofactory and demonstrates key steps in a process that converts simple molecules into a billion-CPU laptop computer.
  • A PowerPoint file containing the animations shown in the galleries.
  • Molecular Machine Part (MMP) and simulation (DPB) files for:
    • The Mark III (K) planetary gear
    • The strained-shell sleeve bearing
    • The Drexler-Merkle differential gear
This is the NanoEngineer-1 1.0.0 Flash Drive as of April 15, 2008.
