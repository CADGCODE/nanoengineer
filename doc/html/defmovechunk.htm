Move

see Move Chunk mode mouse actions

 

 

Move mode can be entered by double clicking the background of either of the Selection modes with the LMB. Move mode can also be entered by pressing the icon from the tools toolbar. Double clicking the window in Move mode, or selecting the will switch the user back to Select Chunks mode.

Separate chunks in other modes will still act as grouped objects. When a chunk is separated it still maintains its bonds. Move mode allows these still bonded objects to be manipulated as different parts. If two chunks are selected they will both move together. When only one chunk is selected the user has the most maneuverability in move mode. A chunk can be rotated or moved in and out of an assembly on an axis. This allows the part to be carefully moved in and out of an assembly for alteration.
