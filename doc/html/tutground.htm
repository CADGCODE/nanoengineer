How to set ground atoms for the simulator.

1. First switch the selection mode to Select Atoms mode .

2. Several atoms must be selected in a chunk.

3. Select the first atom with the LMB, add select the desired amount of additional atoms by holding Shift + LMB.

4. Now that the atom(s) are selected open the jig menu and select Grounds .

5. A properties dialogue is now open.

 

  • Select OK from the property dialogue.
  • You now have grounds added to the part.
