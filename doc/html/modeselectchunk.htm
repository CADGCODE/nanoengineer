Single Click Selection

Single click of LMB selects the atoms or chunks depending on the active selection mode.

 

Bounding Box Selection

 

The LMB selects by dragging a “rectangle” around the atoms or chunks of choice. If a small section is selected in the front of an object, the selection will go right though the object to the back of the screen.

 

Lasso Selection

 

The LMB selects by drawing a circle or “Lasso” around a group of atoms or chunks. If a small section is selected in the front of an object, the selection will go right though the object to the back of the screen.

 

De-Selection

 

The Control key with the LMB de-selects the newly selected portion of the selected atoms.

 

 

Select

Draws a bounding box on the atom or chunks selected.

 

Drag-Lasso Select

 

The LMB can either draw a bounding box or a lasso to select items.

 

Rotate

 

Rotating an object turns the part three dimensionally along its axis.

 

Move

 

The move function displaces a part two dimensionally. This enables the user to move a chunk’s axis to a different origin on screen.

 

Deselect

 

Deselects the selected atoms or chunks.

 

Zoom

 

There are three zoom speeds: fast, slow and normal. This function is activated by using the middle mouse button.

 

Invert Select

 

This function selects new atoms or chunks and deselects the previous selection.

 

 

Context Menus

 

   
  • Select Chunks mode has several functions that can be accessed with the RMB.
  • The context menu offers quick access to switch between atom and chunk selection modes.
  • The context menu offers quick access to the display modes.
  • There are several select features available through the context menu: Connected, Doubly, None, All, and Invert.
  • The Right-Shift context menu offers commonly used modify features such as seperate, stretch, delete, move, and hide.
