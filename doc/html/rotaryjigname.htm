Rotary Motor

 

 

A rotary motor allows the user to apply a torque to the selected atoms. By selecting several atoms of a chunk the user can attach a Rotary Motor to the chunk. The simulator allows user to determine and apply a force to the motor.

See How to insert a Rotary Motor, for a detailed tutorial on inserting a Rotary Motor.
