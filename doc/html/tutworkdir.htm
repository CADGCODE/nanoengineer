How to set your working directory.

1. Open the File drop down menu and select “Set Working Directory…”.

  • This will allow you to navigate your hard drive and find the file where your MMP files are located.

3. Select this file to be set as the directory.

  • Now when you choose the file icon it will open directly to the file set as the working directory.
