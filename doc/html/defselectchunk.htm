Select Chunk

 

see Select Chunk mode mouse actions

 

 

Select Chunk mode allows the user to select whole chunks by clicking the LMB on the desired chunk. Select Chunks mode can also be entered through the RMB with the context menu.

If the LMB is pressed and dragged there are two types of bounded selections. The first selection type is a rectangular bounding box specified by starting and ending points at the corners of a rectangle. The second selection type is a lasso or freehand curve. Both lasso and rectangle will be drawn initially; the program determines through the use of the mouse whether a rectangle or lasso will be used. If the user draws a circle with the mouse, a Lasso will be used. If the user draws a straight line down and a straight line over the rectangle will be used. If the rectangle still appears on the screen it is active; once the rectangle vanishes the lasso will remain and be used. Both methods of selection select the chunk bounded by a box or Lasso.
