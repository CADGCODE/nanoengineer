How to save a file.

1. Use the drop down file menu to select Save As.

2. The File icon will also open the Save As form window if the file isn’t currently saved.

3. This will open your working directory and allow you to determine the type of file to be saved.

4. There four options for saving file in nanoENGINEER-1™.

5. A file can be save as an image (*.jpg), a protein data bank file (*.pdb), a POV-Ray file (*.pov), or the nanoENGINEER-1™ preferred file format of Molecular machine part (*.mmp).

6. Once you have determined the type of file to be saved, select save and the part will be saved in you working directory.
