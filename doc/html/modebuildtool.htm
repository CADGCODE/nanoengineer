Deposit
The add atom function adds a new atom to the open bond. If an open bond is not selected then the add atom function adds an atom to the screen where the user points to and presses the LMB.

 

Rotate

 

Rotating an object turns the part three dimensionally along its axis.

 

Move

 

The move function displaces a part two dimensionally. This enables the user to move a chunk to a different origin on screen.

 

Delete Atom

 

A red sphere appears where the atom is located while in Build mode. Holding control and pressing the LMB will cause the atom to be deleted.

 

Connect open bond

 

Holding shift and pressing the LMB over a open bond will allow the user to move a line from one open bond to another. This will cause the two open bonds to bond.

 

Zoom

 

There are three zoom speeds: fast, slow and normal. This function is activated by using the middle mouse button.

 

Axial Rotation

 

Holding control and moving the mouse from right to left with the MMB pressed will rotate the Chunk in the window.

 

Axial Displacement

 

The build mode has a haze surface which allows a part to be partially alterable. What ever is below the haze is unalterable. The Axial Displacement feature allows the user to zoom the part above the haze layer, or move it below. This function is activated by holding the control key with the MMB pressed, while moving the mouse forward and backwards.

 

 

Context Menus

 

   
  • Build Mode has several functions that can be accessed with the RMB.
  • In Build mode the user is granted quick access to the more commonly used elements and features.
  • The context menu also provides the user a way to switch between Select modes and the Move Chunk mode.
  • The ability to indicate a hotspot is a very important aspect of the Build mode. A hotspot can be set by pressing the RMB and using the context menu to "Set Hotspot".

 

Build Atom mode Dashboard
   
 
  • The radio buttons enable switching between the paste function and the build atom function of the build mode.
  • The first combo box will have a list of the items in the clipboard.
  • The second combo box has a list of elements to choose from.
  • Select the done checkmark when the object is completed. This will move the user back into Select Chunks mode.
