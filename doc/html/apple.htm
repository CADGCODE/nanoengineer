Macintosh Users

The mouse actions pages were written using a Windows three button mouse. This key will help the Apple users translate mouse clicks into keyboard keystrokes.

If you are using a mouse without a scroll button, you will not be able to use the zoom mouse function. There is a zoom window tool provided on the View toolbar. This will allow the user to zoom in on a desired area. There is also way to translate the object in and out of the screen.

Hold the , and move the mouse forward and back.

This will displace the axis of the part into and out of the page. This is different from the scroll zoom function which only moves the view of the part.

Use the Mouse Action Translation Key to translate the Mouse Actions page. The key below has a column on the left with the Apple equivalent to the windows mouse buttons listed in the column on the right.
