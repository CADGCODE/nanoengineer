Tutorial 2

 

Using Cookie Cutter mode to cut a cookie.

  • Before reading this tutorial it may be helpful to read about the Cookie Cutter mode.

1. Once nanoENGINEER-1™ is open, select the Cookie Cutter icon from the tools toolbar. This will bring you into Cookie Cutter mode.

2. Let’s begin by drawing a box around the axis. Hold the LMB and drag to the right, followed by dragging down.

  • This will start a bounding box around the desired area.

3. Once you are satisfied with the size of your shape release the LMB.

  • The thickness of atoms is seen by a spinbox on the dashboard. The default thickness is 2. This represents the number of layers thick the shape will be cut.

4. For our next step we are going to change the thickness in atoms through the spin box located on the dashboard. Change the number 2 in the spin box to a 7.

5. Now draw a second square inside the first square.

  • If you hold the MMB, and drag the mouse you can turn the cookie and see what you have created. This mode has a snap feature built in so it won’t allow you to actually move the cookie, but you can turn it to view you current work.

6. Let’s cut out the center of the inside square. This will hollow the T shaped structure we have created. Hold Ctrl+LMB to bound another box inside the second box that was created. This will subtract the bounded area from the cookie. If the spin box remained unchanged, this will hollow the cookie in the center.

7. Now select the from the dashboard.

  • This will bring you back into Select Chunks mode.

8. The structure you have created still has many open bonds present. The structure should be hydrogenated. With the structure selected in Select Chunks mode select with the LMB from the modify toolbar.

  • Once pressed the open bonds will have hydrogen atoms added to all the open bonds.

9. Now that we have a stable structure, we can select the icon to save the new part.

  • This will open to your working directory and allow you to save the file.

10. Name the file hollowT and select Save.

  • Congratulations, you have just created and saved your first cookie.
