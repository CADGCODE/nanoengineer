nanoENGINEER-1®

Known Bugs/Unsupported Features

All Platforms

The following list of bugs applies to all platforms.

  1. Delete operation cannot be undone. In fact, Undo/Redo are not implemented yet. SAVE YOUR WORK OFTEN!

  2. If you select the part icon (top icon in the model tree) and hit the delete key (or click Delete), the entire part will be deleted without asking the user for confirmation. (147)

  3. Only open one main window. Opening more than one nanoENGINEER-1 application window will have unexpected results.

  4. Dragging and dropping objects onto the part icon at the top of the list places the items(s) at the bottom of the list.

  5. Dragging and dropping items onto a group folder places them at the bottom of the group, not the top.

  6. The front clipping plane will sometimes clip objects that should not be clipped.

  7. Align to common axis averages the axis vectors of the participating chunks. The resultant vector is (usually) different than the axis vectors of any of the chunks.(187)

  8. Any file currently opened by nanoENGINEER-1 can be manually deleted from the computer without any warning message (215)

  9. After drawing a 'cookie' in the cookie cutter, if the orientation view is changed and then any new cookie drawn will be placed at a wrong location. Thus, the user should not change the views while in the cookie cutter. (236)

  10. User can not save exclusive features of nanoENGINEER-1 into the PDB file format. nanoENGINEER-1 doesn't give any warning message for users trying to save such features in a .pdb file. (252)

  11. There is a display bug associated with a cut operation on a chunk. It stretches the interchunk bond and keeps bonded portion of cut chunk visible.

  12. Extrude Mode's 'extrude a ring' feature has some bugs. These will be fixed for the Beta release of nanoENGINEER-1. To use extrude's 'Ring Mode' effectively, make sure you follow these steps:

    1. Get everything adjusted first in the 'rod mode'. For example, adjust the number of copies (N), check boxes on the dashboard about the final product and the way you want bonding between chunks to form.

    2. Then you may switch to "Rod Mode" by selecting ring from the rod pulldown menu on the dashboard. While viewing the ring, you may increase the number of copies.

    3. If you need to do any further adjustments, switch back to "Rod Mode", make changes and switch back to the "Ring Mode".

    4. Select the 'Done' button on the dashboard when finished. (155)

  13. When starting a new part or opening a new file, the object name indexing is not reset to 1. For example, new chunk names continue to increment from the previous part's index number. (64)

  14. The MMP File Format specification included in the nanoENGINEER-1 Assistant is obsolete. The most current version can be found at www.nanorex.com/mmpformat.htm

The following list of unsupported features applies to all platforms.

  1. Undo/Redo.

  2. When moving a chunk, any rotary and/or linear motors attached to it will not move with the chunk. (166)

  3. Cannot change color of individual atoms.(189)

  4. Can not move, translate or rotate a chunk in a precise manner (e.g. by editing the cartesian or cylindrical coordinates) (9)

  5. Reference axis is implemented but the reference planes for modeling purpose is not implemented yet. (47)

  6. Simulator will support bonding between elements specified in the element selector of nanoENGINEER-1. Multiple bonding is not supported for alpha.

  7. The 'Clipboard' has limitations and some bugs.

  8. Do not drag and drop anything between the model tree and the clipboard.

  9. The symmetry information for a selected chunk cannot be obtained. This is planned for future release of the software.(261)

Windows Only

The following list applies to Win32 systems only.

  1. In the Cookie Cutter, Shift+Left only deletes drawing that is outside but doesn't draw inside the lasso/ rectangle (190)

  2. During drag-and-drop, node icons are not shown in the cursor's drag object.

Macintosh Only

The following list applies to Macintosh only.

  1. Sometimes a cursor's bitmask will suddenly not work, leaving a white square around the cursor.

Linux Only

  1. In the Cookie Cutter, Shift+Left deletes items outside the selection, but doesn't draw inside the lasso/ rectangle (190)
