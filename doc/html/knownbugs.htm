nanoENGINEER-1®

Known Bugs/Unsupported Features

All Platforms

The following list of bugs applies to all platforms.

  1. Delete operation cannot be undone. In fact, Undo/Redo are not implemented yet. SAVE YOUR WORK OFTEN!
  2. If you select the part icon (top icon in the model tree) and hit the delete key (or click Delete), the entire part will be deleted without asking the user for confirmation. (147)
  3. During Minimize, the progress bar will stay at 0% until the last moment, then jumps to 100% before hiding.
  4. Only open one main window. Opening more than one nanoENGINEER-1 application window will have unexpected results.
  5. Dragging and dropping objects onto the part icon at the top of the list places the items(s) at the bottom of the list.
  6. Dragging and dropping items onto a group folder places them at the bottom of the group, not the top.
  7. Dragging and dropping items between the clipboard and model tree can sometimes have unexpected results. Use cut, copy and paste instead.
  8. The front clipping plane will sometimes clip objects that should not be clipped.
  9. Align to common axis averages the axis vectors of the selected chunks. The resultant vector is (usually) different than the axis vectors of any of the chunks. (187)
  10. Any file currently opened by nanoENGINEER-1 can be manually deleted from the computer without any warning message. (215)
  11. After drawing a 'cookie' in the cookie cutter, if the orientation view is changed and then any new cookie drawn will be placed at a wrong location. Thus, the user should not change the views while in the cookie cutter. (236)
  12. User can not save exclusive features of nanoENGINEER-1 into the PDB file format. nanoENGINEER-1 doesn't give any warning message for users trying to save such features in a .pdb file. (252)
  13. There is a display bug associated with a cut operation on a chunk. It stretches the interchunk bond and keeps bonded portion of cut chunk visible. If this happens, select the visible chunk and use "Delete Bonds" to remove the bonds between the two chunks.
  14. Extrude Mode's 'extrude a ring' feature has some bugs. These will be fixed for the Beta release of nanoENGINEER-1. To use extrude's 'Ring Mode' effectively, make sure you follow these steps:
    1. Get everything adjusted first in the 'rod mode'. For example, adjust the number of copies (N), check boxes on the dashboard about the final product and the way you want bonding between chunks to form.
    2. Then you may switch to "Rod Mode" by selecting ring from the rod pulldown menu on the dashboard. While viewing the ring, you may increase the number of copies.
    3. If you need to do any further adjustments, switch back to "Rod Mode", make changes and switch back to the "Ring Mode".
    4. Select the 'Done' button on the dashboard when finished. (155)
  15. When starting a new part or opening a new file, the object name indexing is not reset to 1. For example, new chunk names continue to increment from the previous part's index number. (64)
  16. The MMP File Format specification included in the nanoENGINEER-1 Assistant is obsolete. The most current version can be found at www.nanorex.com/mmpformat.htm
  17. Rotary and Linear motors do not move with the chunks they are attached to. (168)
  18. Cannot select a motor, ground, thermostat or thermometer in the GLpane. These may only be selected via the model tree. (6)
  19. "Select Doubly" can take a long time without indication to the user that it is still working.
  20. "Hint Cursors" can sometimes get stuck on. Pressing Shift or Cntl/Cmd will usually unstick the cursor. (113, 158)
  21. Build Mode: The skull and crossbones cursor will not display (when pressing Ctrl/Cmd) after clicking in the Model Tree area. (217)
  22. Cookie Mode: Changing views (Front, Top, Right, etc.) and then making a selection does not select properly. (236, 277)
  23. Cookie Mode: Cannot change the thickness of the current layer once selection has started.
  24. Cookie Mode: When adding a new layer, there is no visual cue indicating a new layer has started. (281)
  25. Cookie Mode: Large selections can cause response latency.
  26. Larger atoms can obscure open bonds when displayed in VdW mode. (258)
  27. Movie Player: If you immediately select the Movie Player after Minimize, you will get the error message "Movie file [/homedir/Nanorex/sim-PID#.xyz] not valid for the current part".
  28. Simulator: The parameters in the simulator always reset to 900 frames, 10 steps per frame and 300K rather than the last parameters entered from the previous simulation run.
  29. The Zoom Tool is still experimental. Some Mac users report cosmetic bugs. It is a good idea to save your work before using it.

 

The following list of unsupported features applies to all platforms.

  1. Undo/Redo not implemented.
  2. Cannot change color of individual atoms.(189)
  3. Can not move, translate or rotate a chunk by editing its coordinates or providing an x, y, z offset. (9)
  4. Datum objects (points, lines, and planes) are not implemented yet. (47)
  5. Simulator will support bonding between elements specified in the element selector of nanoENGINEER-1. Multiple bonding is not supported for alpha.
  6. The 'Clipboard' has limitations and some bugs.
  7. Do not drag and drop anything between the model tree and the clipboard.
  8. The symmetry information for a selected chunk cannot be obtained. (261)
  9. sp and sp2 bonds are not supported.
  10. aromatic bonds are not supported.
  11. Bonds are not selectable.

 

Windows Only

The following list applies to Win32 systems only.

  1. In the Cookie Cutter, Shift+Left only deletes drawing that is outside but doesn't draw inside the lasso/ rectangle (190)
  2. During drag-and-drop, node icons are not shown in the cursor's drag object.

 

Macintosh Only

The following list applies to Macintosh only.

  1. Sometimes a cursor's bitmask will suddenly not work, leaving a white square around the cursor.

 

Linux Only

  1. In the Cookie Cutter, Shift+Left deletes items outside the selection, but doesn't draw inside the lasso/ rectangle (190)
