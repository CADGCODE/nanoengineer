How to select individual atoms.

1. There are two types of selection in nanoENGINEER-1™.

2. Hold the RMB and use the context menu to choose either Select Atoms or Select Chunks.

  • This can also be done by selecting the icon for Select Atoms mode.

3. Pressing the LMB over a atom in Select Atoms mode will select the clicked atom.

4. Holding the LMB and dragging a bounding box around a desired section of atoms will select the atoms bounded by the box.

5. A lasso option for selecting atoms appears within the bounding box. If the user draws a circle around the atoms to be selected while holding the LMB this will also select the bounded atoms.
