Using the Simulator.

1. Once an object is selected, pressing the simulator icon will open a form window.

2. The window allows the user to determine time step, temperature, steps per frame, and total number of frames.

3. Choosing the Save File button will create an (*.xyz) file, this is a text file with positional information. The Save File option will open your working directory and allow the user to save the trajectory (*.xyz) file by default. There is also a file format choice in the save form. The save file form gives the choice between saving as a (*.dpb) file and an (*.xyz) format. The (*.dpb) format is a playable movie file, and the (*.xyz) is a text file with positional information.

4. Selecting Create Movie will create a (*.dpb) file that is a playable movie file.

  • In order to observe the simulation for a longer time interval increase the Timestep to 300 (hundredths of femtosecond).

4. Select Create Movie from the form window.

  • The simulator has now prepared the simulated movie file.

5. A progress menu will appear displaying duration of time for the simulation and the progress in a percentage.

5. Select the Movie icon to view the simulated movie file in the main window.
