MINIMUM SYSTEM REQUIREMENTS:

Linux:
1 GHz Intel/AMD processor or equivalent
Mandrake 10.0 or higher
512 MB RAM
300 MB available disk space
3D graphics accelerator card

Macintosh:
1 GHz PowerPC G4 processor
Mac OS X 10.3 and later
512 MB RAM
300 MB available disk space
3D graphics accelerator card

Windows:
1 GHz Intel Pentium M processor or equivalent
WindowsXP
512 MB RAM
300 MB available disk space
3D graphics accelerator card

INSTALLATION:

Linux:
Get the ne-1a.rpm package installer and simply run it.

MacOS:
Get the ne-1a.pkg package installer and simply run it.

Windows:
Get the setup.exe package installer and simply run it.
