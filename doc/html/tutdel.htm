Delete atoms and chunks.

 

1. The delete key on the user’s keyboard will delete the selected atom or chunk.

  • The icon in the Edit tool bar will also delete the selected parts.

2. Items can also be selected and deleted with the model tree.

  • In the model tree the user can select an object and use any of the previously discussed methods to delete the object.
