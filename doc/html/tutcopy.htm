Using the copy and paste function.

 

1. Once an object is selected it can be copied by selecting the copy icon from the Edit toolbar.

  • A copied object is added to the clipboard.

2. Once an object is on the clipboard, you can select paste icon and nanoENGINEER-1™ will move the user to Build Atom mode.

3. Selecting the paste icon again or holding CTRL + V will now paste the copied object into Build Atom mode.

4. There are two ways in build mode to bond the copied and pasted objects together.

  • Holding shift and dragging the LMB from one open bond to another will cause the two open bonds to bond to each other.
  • The second way to bond two chunks together is to set a hotspot.

5. Right click over an open bond.

6. In the context menu is a “Set Hotspot” selection choice.

  • Once a open bond is set as a hotspot the next LMB click will paste will bond at the designated hotspot.
