Open

Ctrl+O

 

 

The Open icon is a pathway to the files and available libraries on your P.C.. This function enables the user to open a file and select a PDB or MMP file for alteration.

 

Save

Ctrl+S

 

 

The Save icon will save altered specifications to a pre-existing File.
