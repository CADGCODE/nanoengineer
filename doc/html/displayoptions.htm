Display Options

 

Display Hierarchy

 

 

 

nanoENGINEER-1™ has a display setting hierarchy. An object that is created in one mode will switch its display setting when entering a different mode. When an object enters the Select modes the display will switch from the current modes default display setting into a VdW setting. Each mode has a default display setting. Once the user sets the display of a chunk or atom the setting won’t switch when the mode switches.

The display setting of an individual atom set in Select Atoms mode has first priority over an atoms displayed view. When an individual atoms display is set in Select Atoms mode the default display of the atom is changed to the new setting. When attempting to change the chunks display setting the user can not override settings designated for individual atoms. The user must switch to Select Atoms mode to change the display setting of the atom. This hierarchy extends to display settings with unselected chunks altered in the main window. When a display setting is applied to the unselected chunks in the main window all chunks with unaltered display settings will change their display. Any chunk or atom with previously designated settings set with the Select modes will not change their display with the unaltered parts in the window.

 

.

 

 

 

 

Default

 

 

 

Resets the display of the selected atoms or chunks to their default display mode.

 

Invisible

 

 

 

Changes the display of the selected atoms or chunks to invisible, making them invisible, but does not delete them.

 

Lines

 

 

 

Changes the display of the selected atoms or chunks to Lines.

 

Tubes

 

 

 

Changes the display of the selected atoms or chunks to Tubes.

CPK

 

 

 

Changes the display of the selected atoms to CPK mode, also known as "Ball and Sticks" mode. Atoms are rendered as spheres and bonds are rendered as grey cylinders.

 

VDW

 

 

 

Changes the display of the selected atoms to Van der Waals mode. Atoms are rendered as spheres with a size equal to the VdW radius. Bonds are not rendered.

 

Chunk Color  

 

 

When in select chunks mode the user can change a selected chunks color. Select the chunk and choose the chunk color form from the drop down display menu. A color palette will open allowing the user to choose a different color.

 

Reset Chunk Color  

 

 

Resetting the chunk color returns the color of the chunk to it's default settings.

 

Background Color  

 

 

The backround color of the main window can be altered through the display drop down menu. A color palette will open allowing the user to choose a different color for the main window's background.

 

Set Atom Colors to default  

 

 

nanoENGINEER-1™ has a default setting of atom colors that the program uses upon being opened. There is an alternate set of colors as well. These two selections, default and alternate operate as an inverse of each other. This inverse relationship allows the user to switch back and forth between color sets.

 

Set Atom Colors to Alternate  

 

 

nanoENGINEER-1™ has a alternate setting of atom colors as well as the default settings. These two selections, default and alternate operate as an inverse of each other. This inverse relationship allows the user to switch back and forth between color sets.
