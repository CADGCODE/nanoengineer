How to open a part from the parts libraries.

 

1. The Open File icon can be found in the FILE drop down menu, or by pressing CTRL + O.

  • A part can also be opened by selecting the file folder icon on the File Toolbar.

2. Select the File folder icon from the File Toolbar.

  • The File folder icon will allow you to navigate the files on your hard drive and find a MMP file.

3. Select the MMP file of interest and double click the file to open the file in nanoENGINEER-1™.
