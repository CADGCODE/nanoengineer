Using Select Atoms mode

 

see Select Chunk mode mouse actions

 

 

The Select Atoms mode allows the user select single atoms with one click of the LMB. Select Atoms mode can also be entered through the RMB with the context menu.

A large section of atoms can be selected by drawing a bounding box around a region of atoms. If the LMB is pressed and dragged there are two types of bounded selections. The first selection type is a rectangular bounding box specified by starting and ending points at the corners of a rectangle. The second type of selection is a lasso or freehand curve. Both lasso and rectangle will be drawn initially; the program determines through the use of the mouse whether a rectangle or lasso will be used. If the user draws a circle with the mouse, a Lasso will be used. If the user draws a straight line down and a straight line over the rectangle will be used. If the rectangle still appears on the screen it is active; once the rectangle vanishes the lasso will remain and be used. Both methods of selection select the atom bounded by a box or Lasso.

In Select Atoms mode if the display mode of an atom is changed the default display of the atom is changed. This will cause the selected and altered atom to remain in the new display mode until the atom is selected and the default key is pressed.
