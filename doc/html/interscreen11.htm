Minimize  

 

 

Arranges the atoms of the selected chunk(s) to their chemically stable point of equilibrium in reference to the other atoms in the structure.

 

Hydrogenate

 

 

 

Adds hydrogen atoms to all the open bonds in the selection.

 

Dehydrogenate

 

 

 

Removes all hydrogen atoms from the selection.

 

Passivate

 

 

 

nanoENGINEER1™ has a feature that removes the open bonds of a chunk and then makes the structure chemically correct based on the number of bonds formed.

1. Passivate first checks for the number of bonds the selected element has formed.

2. It treats the selected atom as if there were no open bonds present.

3. Passivate then converts the atom into an element which has the same number of bonds as the previous element without the leftover open bonds.

Change Element

 

 

 

Allows you to change the element type of selected atoms. When an atom is selected choose the new element and press the transmute key to change to the new atom. Change Element can also be used in Build Atom mode to change atom types.

The element selector has a check box that will allow the user to transmute incorrectly bonded atoms. If the user desires to swap a hydrogen for a properly bonded carbon, you can check this box and transmute the atom into the chunk. This will occur with a warning in the message bar indicating the improper bonding. The atom's bounding box will also appear as red instead of the correctly bonded atoms blue bounding box.

 

Stretch

 

 

 

Stretches the bonds of the selected chunk(s).

 

Separate

 

 

 

Creates a new chunk from the currently selected atoms. A seperated chunk is still bonded to the chunk it was seperated from. The seperate function allows a chunk to be seperated and treated as two seperate pieces. The bonds will only be invisible while the displayed view is VdW.

 

Weld Chunks

 

 

 

Merges two or more chunks into one chunk when in Select Chunks mode. In Select Atoms mode the weld feature creates a new chunk from a section of atoms.

 

 

Align to common axis

 

 

 

Automatically aligns two or more chunks to a common axis. You must first select two or more chunks before using this feature.
