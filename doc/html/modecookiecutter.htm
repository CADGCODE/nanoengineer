Cookie Cutter Surfaces

 

 

When using the Cookie Cutter mode you have a choice of three different diamond lattices to choose from. (The Orientation views are only applicable to the 100 Surface. If you select the top orientation the surface will automatically switch the surface to the 100 lattice.)

The surface orientation keys will reorient an object in Select mode to the selected orientation.

 

100 Surface

 

 

The viewpoint orientation of this surface is set at (1, 0, 0) plane in Cartesian coordinates.

 

110 Surface

 

 

The viewpoint orientation of this surface is set at (1, 1, 0) plane in Cartesian coordinates

 

111 Surface

 

 

 

The viewpoint orientation of this surface is set at (1, 1, 1) plane in Cartesian coordinates.

 

 

 

Draw/Drag Selection Box

 

This feature is available in cookie cutter mode. Pressing the LMB starts the first point of a line. The second LMB click draws the line on the screen. If several lines are used to bind an area in Cookie cutter mode the area will be selected as a cookie. The LMB can also operate as a Drag bounding box or Lasso selection option.

 

Rotate

 

The rotate function enables the user to turn part three dimensionally along its axis.

 

Move

 

The move function displaces a part two dimensionally. This enables the user to move a chunk ’s axis to a different origin on screen.

 

Subtract Select

 

 

Subtracts the new selection from the originaly selected area.

 

Zoom

 

There are three zoom speeds: fast, slow and normal. This function is activated by using the middle mouse button.

 

Select Current

 

This function deselects all but the currently selected section of a cookie.

 

Drag-Lasso Select

This function deletes the selected atoms or chunks.

 

Axial Rotate

 

Holding control and moving the mouse from right to left with the MMB pressed will rotate the Chunk in the window.

 

Axial Displacement

 

Moving the mouse forward or back while holding control and MMB, will zoom all the objects in view in and out of the page. This feature displaces the axial origin of a chunk.

 

 

 

Context Menus

 

   
 
  • Cookie Cutter mode has several functions that can be accessed with the RMB.
  • The user can Start Over, Back up, Done, or Cancel the chunk they are working on.
  • The layer feature in Cookie Cutter mode lets the user to push the layer under the grid, this allows a second layer to be placed on top of the first.

Build Atom mode Dashboard

   
 
  • Surface orientations can be selected through the dashboard in Cookie cutter mode.
  • The add layer tool allows the user to move the current cookie below the grid.
  • The form window allows the user to specify in angstroms the thickness of the diamond lattice.
  • A spin box is provided to input integer values of the thickness in layers of diamond.
  • The user is also provided with a Back, Start Over, Done and Cancel key located on the far right of the dashboard.
