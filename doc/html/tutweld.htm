How to weld chunks together.

1. For this feature you need two chunks in the window.

2. Use the copy function to create a second chunk from your original chunk. See " Using the copy and paste function ".

3. First select both chunks by selecting the Select Chunks mode icon .

4. Press the LMB on the first chunk.

5. Then press Shift + LMB to add-select the second chunk.

6. Now that both separate chunks are selected, select the Weld Chunk icon .

  • The two chunks are now grouped as one chunk.
