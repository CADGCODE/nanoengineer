Move

The move function displaces a part two dimensionally. This enables the user to move a chunk’s axis to a different origin on screen.

 

Rotate

 

Rotating an object turns the part three dimensionally along its axis.

 

Move

 

The move function displaces a part two dimensionally. This enables the user to move a chunk’s axis to a different origin on screen.

 

3-D motion

 

Holding the MMB and moving the mouse allows the user to turn the selected chunk with 3 degrees of freedom.

 

 

Zoom

 

There are three zoom speeds: fast, slow and normal. This function is activated by using the middle mouse button.

 

Constrained Motion

 

This feature moves the chunk along its axis.

 

Axial Rotate

 

Holding control and moving the mouse from right to left with the MMB pressed will rotate the Chunk in the window.

 

Axial Displacement

 

Moving the mouse forward or back while holding control and MMB, will zoom all the objects in view in and out of the page. This feature displaces the axial origin of a chunk.

 

 

 

Context Menus

 

   
  • Move mode has several functions that can be accessed with the RMB.
  • The context menus offer quick access to all the display modes.
  • There are several commonly used features located in the Move Chunk mode context menu.
  • The context menu also provides the user a way to switch between Select modes and the Move Chunk mode.

 

Move Chunk Dashboard
   
 
  • Select the Done Icon on the dashboard to reenter Select Chunks mode.
  • Double clicking the LMB on the screen in Move Chunk mode will move you into Select Chunks mode.
  • Double clicking the LMB on the screen in either Select mode will bring you into Move mode.
