Tutorial 1

 

Creating a piece of adamantine.

  • Before reading this tutorial it may be helpful to read about build mode.
  • Build Atom mode can create chunks and deposit atoms to a chunk.
  • Build atom mode has three ways of choosing the type of atoms to deposit. The Dashboard combo box , the Change Element Icon , and by holding Alt and pressing the first letter of an elements name.
  • Carbon is the default atom for Build mode.

 

Creating a piece of adamantine.

1. Click on the Build Atom icon .

2. Start by clicking LMB once on the screen.

  • Now there is one carbon atom present, represented in tube mode (the default display setting in Build mode).

3. Move the curser over the red open bond (open bond), and wait till the open bond lights up.

 

4. Press the LMB to bond the second carbon atom to the first.

  • What is on screen now should represent a dumbbell, with three red open bonds on each end.

5. Move the curser over another open bond, and press the LMB to deposit the third atom.

6. Rotate the structure with the MMB to reposition the three carbon structure on screen and begin to add more atoms. See Build mode mouse actions for more information on how interact with the Build Atom mode.

7. Now add the fourth carbon atom to the structure by lighting the open bond and press the LMB.

8. The next step is to add the fifth carbon atom by moving the curser over the open bond, and press the LMB.

9. Add the sixth carbon atom to close off the hexagonal ring.

  • This hexagonal ring has six total carbon atoms. If you make an error, you can delete atoms by Ctrl + LMB. See Build mode mouse actions for more information on how interact with the Build Atom mode.

 

10. Use the MMB to reposition the ring.

  • You can also zoom in and out using the scroll function of the MMB. See Build mode mouse actions for more information on how interact with the Build mode.
  • You should now see a hexagonal ring on the screen.

11. Add another carbon atom to the ring at the point where the open bonds make a triangle, the triangle can be found on either side of the ring.

12. The carbon atom will look like this when added to a open bond on the triangle.

13. Add another carbon to a second point of the triangle.

14. Add a third carbon to the last point on the triangle.

  • Your structure is almost complete, and should resemble a cage that is not closed yet.

15. Select one of the open bonds in the center of the most recently added three atoms. Move the cursor till the bond lights up, and deposit the last carbon atom to create your piece of adamantine.

  • Your structure should look like this.

16. Select the done check mark from the dashboard.

17. This will bring you back into Select Chunk mode.

18. Select the file icon from the File toolbar. This allows the user to save there work.

19. Name the part “Adamantine”, and save it as a (*.mmp) file.
