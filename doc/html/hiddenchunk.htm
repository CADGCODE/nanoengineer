Hidden Chunk

 

 

 

 

The user can choose to hide or unhide a chunk. A hidden chunk will be inaccessible and absent from view in the main window. A hidden chunk will appear blacked out in the model tree. To reanimate the chunk, select unhide in the model tree. Unhide is located in a context menu found by pressing the RMB over the selected chunks name in the model tree.
