Changing the displayed view of atoms.

 

1. The display icons at the top of the screen that will change the selected items to the new display mode choice.

  • The display mode selections can also be found in the Display menu pull down menu, or by right clicking on the chunk and using the context menu.

2. First select the chunk or atoms you want to change.

3. Then once the chunk(s) or atom(s) are selected, select a different display mode from the Display toolbar at the top of the screen.
