 Movie

 

 

 

Movie mode is where the user will view & play the trajectory (*.dpb) file that was created by the Simulator. This mode will show the simulation created by the simulated parameters.

Using the spin box on the right of the frame window the user can move frame by frame while viewing the movie. The user can add atoms to the chunk by pausing the playback. Once the simulation is paused the user can use any of nanoENINEER-1™’s methods to add atoms. If atoms have been added to the chunk the user must re-simulate to play the movie file.

The movie file can be played forward and backwards. The user is offered a save icon and folder choice to aid in saving the simulated movie.

The information icon will display the frame and timestep information about the movie in the history bar.

Movie Player Dashboard

Alpha users:

In the current alpha release the user can pause the movie. Below are a list of permitted actions while the movie is paused, as well as a list of actions that will cause the movie player to malfunction.

Here is a list of things that will not affect the movie's "validity":

  • File: Save, Save As
  • Edit: Copy
  • View: All
  • Display: All
  • Modify: Separate, Weld (because these do not change the atom positions)
  • Tools: Select Atoms, Select Chunks
  • GLPane: zoom in/out, rotate, select/unselect
  • Model Tree: Hide/Unhide, Copy, Properties, change order of nodes

 

List of things that will make the movie invalid:

  • File: Open, Close, Insert
  • Edit: Cut, Paste, Delete
  • Jigs: all jigs will cause malfunctions.
  • Modify: Minimize, Hydrogenate, Dehydrogenate, Passivate, Element (Transmute), Stretch, Align to Common Axis
  • Tools: Move Chunks, Build, Cookie Cutter, Extrude (unless you do nothing while in the tool/mode) and Simulate (this will overwrite the moviefile).
  • Main Window : select + delete
  • Model Tree: Cut, Delete
