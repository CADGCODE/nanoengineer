How to insert a Thermostat?

1. First switch the selection mode to Select Atoms mode .

2. Several atoms must be selected in a chunk.

3. Select the first atom with the LMB, add select the desired amount of additional atoms by holding Shift + LMB.

4. Now that the atom(s) are selected open the jig menu and select Thermosat.

5. A Form window is now open to the user.

6. There are many editable options in the form window, for simplicity use the default options by selecting OK.

  • You now have a Thermostat added for the simulator.
