Tools Toolbar

 

12

 

 

 

Select Chunk

 

 

 

Activates Select Chunks mode, allowing you to select chunks with the mouse.

 

 

 

Select Atoms

 

 

 

Activates Select Atoms mode, allowing you to select atoms with the mouse.

 

 

Move Chunks

 

 

 

Activates Move Chunks mode, allowing you to select, move and rotate individual chunks with the mouse.

 

 

Build Atom

 

 

 

Activates Build mode, allowing you to build chunks with the mouse.

 

 

Cookie Cutter

 

 

 

Activates Cookie Cutter mode, allowing you to cut out 3-D shapes from a slab of diamond lattice.

 

 

Extrude

 

 

Activates Extrude mode, allowing you to create a rod or ring using a chunk as a repeating unit.

 

 

Simulator

 

 

 

Creates a (*.dpb) trajectory (movie) file by calculating the inter-atomic potentials and bonding of the entire model. The user determines the number of frames in the movie, the time step, and the temperature for the simulation.

 

 

 

Movie

 

 

 

Plays the most recent (*.dpb) trajectory (movie) file created by the Simulator.
