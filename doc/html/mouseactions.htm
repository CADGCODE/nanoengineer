Mouse Control Definitions

Left Mouse

Button

(LMB)

 

 

 

 

Holding down the Left Mouse Button (LMB) will activate the function linked to the mode in use.

Middle Mouse Button

(MMB)

 

 

 

 

Holding down the Middle Mouse Button (MMB) will activate the function linked to the mode in use.

Right Mouse

Button

(RMB)

 

 

 

 

Holding down the Right Mouse Button (RMB) will activate the function linked to the mode in use.

Scroll Middle

Mouse Button

(SMMB)

 

 

 

 

Scroll middle mouse button (SMMB) is a zoom function of the program. There are three speeds that can be used for this function.

 

Drag Left

Mouse Button

 

 

 

Holding the Left Mouse Button is held down while moving the mouse in any direction will activate the drag function connected with the mouse button.

Drag Middle

Mouse Button

 

 

 

Holding the Middle Mouse Button down while moving the mouse in any direction will activate the drag function connected with the mouse button.

 

Drag Right

Mouse Button

 

 

 

 

Holding the Right Mouse Button is held down while moving the mouse in any direction will activate the drag function connected with the mouse button.

 

Select

Once an item is selected a bounding box drawn on the atom or chunk selected.

 

Drag-Lasso Select

The LMB can either pull down a bounding box or a circle a lasso to select items.

 

Draw-Drag

Selection Box

This feature is available in the cookie cutter mode. Clicking the LMB starts the first point of a line, as the second point is clicked a line appears. If several lines are used to bind an area in Cookie Cutter mode the area will be selected as a cookie. The LMB can also operate as a drag bounding box or Lasso selection option.

 

Rotate

 

The rotate function enables the user to turn part three dimensionally along its axis.

 

Move

 

The move function displaces a part two dimensionally. This enables the user to move a chunk’s axis to a different origin on screen.

 

 

Add Atom

 

The add atom function adds a new atom to the open bond, bonding the two atoms together. If a open bond is not selected then the add atom function adds an atom to the screen where the user points and clicks.

 

Deselect

 

The Deselect function deselects the selected atoms or chunks.

 

Constrained

Motion

 

This feature moves the chunk along its axis.

 

Zoom

 

There are three zoom speeds: fast, slow and normal. This function is activated by using the middle mouse button.

 

Select Current

 

This function deselects all but the current selection of atoms or chunks.

 

Delete Selected

This function deselects the all but the current selection of atoms or chunks.

 

Toggle Select

This function selects new atoms or chunks and deselects the previous selection.
