About nanoENGINEER-1™

nanoENGINEER-1™ is a 3-D molecular engineering software platform focused exclusively on electro-mechanical nanoengineering.

nanoENGINEER-1™ includes a molecular design module that combines capabilities found in traditional molecular modeling packages with features found in popular 3-D mechanical CAD systems. Atom also includes a sophisticated molecular dynamics simulation module. This module is integrated with the design interface allowing users to run simulations of complex nanosystems in order to verify performance of designs. This tight coupling between design and simulation allows the user to seamlessly design and analyze nanosystems within the same application environment.

nanoENGINEER-1™ is supported on the following platforms:

  • MS/Windows -- Windows XP
  • Unix/X11 -- Linux
  • Macintosh -- Mac OS X Version 10.2.8

nanoENGINEER-1™ is released in different editions:

  • nanoENGINEER-1™ Personal Edition
  • nanoENGINEER-1™ Free Edition
