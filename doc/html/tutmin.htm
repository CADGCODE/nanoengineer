Using the minimize function.

1. Select the minimize icon .

  • This feature does not operate in Cookie Cutter mode or Extrude mode.

2. Once the minimize icon is selected you will see a progress bar appear.

3. Once the progress bar has reached 100% the chunk will attain it's minimum structure.
