Drag Extrusion
Three copies are in the main window when extrude mode is entered. Selecting the outermost copy with the LMB allows the user to move all three copies simultaneously to line up the open bonds for bonding.

 

Rotate

 

Rotating an object turns the part three dimensionally along its axis.

 

Move

 

The move function displaces a part two dimensionally. This enables the user to move a chunk to a different origin on screen.

 

Zoom

 

There are three zoom speeds: fast, slow and normal. This function is activated by using the middle mouse button.

 

Axial Rotate

 

Holding control and moving the mouse from right to left with the MMB pressed will rotate the Chunk in the window.

 

Axial Displacement

 

Moving the mouse forward or back while holding control and MMB, will zoom all the objects in view in and out of the page. This feature displaces the axial origin of a chunk.

 

 

 

Context Menus

 

   
  • The ability to change the displayed view of the extrusion is the context menu.
  • The user also has a start over and cancel option in the context menu.

 

Build Atom mode Dashboard
   
 
  • The radio buttons have two sections in the extrude mode. "Show" determines what is visible in the extrude mode. "When done" determines what occurs when the user reenters select chunk mode.
  • The tolerance scroll bar will enlarge or shrink the bond offset spheres.
  • The X,Y,Z, spin boxes provide a way to specify the distance between extrusions.
  • The ring/rod combo box enables the user to switch between the ring and rod mode of the extrusion tool.
  • The user is also provided with a Back, Start Over, Done and Cancel key located on the far right of the dashboard.
