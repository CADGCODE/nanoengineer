Getting Started

 

The assistant help desk is designed to be a guided tour of nanoENGINEER-1™. To get the best use out the assistant, we recomend you read the overviews carefully. Once the user has read through the overviews, the tutorials will give the hands on experience needed to get started creating molecular machine parts.

 

The Assistant help desk is split into two sections. The first section is a detailed overview of all aspects of nanoENGINEER-1™.

1. nanoENGINEER-1™ user interface navigation

  • A picture of the interface is divided into subsections and labeled with numbers. Each numbered area has a section link located in a table below the picture. Select the section of interest for a detailed explanation.

2. Using the modes

  • Each tool has an associated mode. This overview is a detailed description of the mode associated with each tool.

3. Mouse Actions

  • Every mode has different features that can be activated with different key and mouse combinations. The combination Ctrl + LMB will activate a mode specific feature in Move Chunk mode. The same Ctrl + LMB will activate different features in Cookie Cutter mode. The mouse actions overview is a detailed explanation of the mouse actions in each mode.

4. Keyboard accelerators

  • There are many shortcuts that can be used in nanoENGINEER-1™. The alt key is linked to many of nanoENGINEER-1™' s features. The keyboard accelerator page has a detailed table of all the shortcuts in nanoENGINEER-1™.

5. The model tree navigation

  • A picture of the Model Tree is divided into subsections which are labeled by numbers. Each numbered area has a section link located in a table below the picture. Select the section of interest for a detailed explanation.

6. MMP File format

  • The MMP file format is the standard format for nanoENGINEER-1™. This overview tells the user the type of information that is stored in the file.

 

Section Two of the Assistant help desk is a collection of detailed tutorials. There are three full tutorials to familiarize the user with nanoENGINEER-1™. There is also a list of general function tutorials found under more tutorials.

 

1. Tutorial 1

  • The first tutorial teaches the user how to create a piece adamantine using the Build Atom tool. Once the part is created the user learns how to save the newly created part.

2. Tutorial 2

  • The second tutorial teaches the user to create cookies using the Cookie Cutter tool. The user creates a cookie in the shape of a hollow T. The user will then hydrogenate the newly created cookie and save the cookie.

3. Tutorial 3

  • The third tutorial teaches the user how to create a planet gear using the Extrude tool. In this tutorial the user opens moiety from the part library. Tutorial 3 guides the user through the extrusion and hydrogenation process in Extrude mode.

The Assistant help desk is best used with a detailed tour of the overviews, followed by working through the tutorials. Tutorials one, two, and three will help to familiarize the user with the more complicated tools of nanoENGINEER-1™.
