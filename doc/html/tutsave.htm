How to save a file

1. Use the drop down file menu to select Save As.

  • The file icon will also open the Save As form window if the file isn’t currently saved.
  • This will open your working directory and allow you to determine the type of file to be saved.

  • There five options for saving file in nanoENGINEER-1™.
  • A file can be saved as an image (*.jpg), a protein data bank file (*.pdb), a POV-Ray file (*.pov), Model mdl (*.mdl),or the nanoENGINEER-1™ preferred file format of Molecular machine part (*.mmp).

 

2. Once you have determined the type of file to be saved and named it, select "Save" and the part will be saved in you working directory.
