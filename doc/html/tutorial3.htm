Tutorial 3

 

Creating a Planet Gear with the Extrude tool.

  • Reading the Extrude mode explanation prior to this tutorial may be helpful.

1. Open the file folder icon .

 

2. Select the path partlib/mmplib/PlanetGearMoiety.mmp

 

3. Select the planet gear moiety.

 

4. With the moiety selected click the Extrude tool icon .

 

5. This opens Extrude mode with three copies of the moiety.

 

6. Select CPK from the display toolbar.

 

  • This will switch the moieties view to CPK , which allows you to see the bonding between open bonds more accurately.

 

7. By default the chunks will be separated a fixed difference that can be altered through the dashboard at the bottom of the screen. The chunks can also be moved manually by dragging the outermost chunk with the LMB.

  • The user has four form spin boxes to move the placement of the chunks in reference each chunk.
  • The X spinbox determines the spacing in the X plane.
  • The Y spinbox determines the spacing in the Y plane.
  • The Z spinbox determines the spacing in the Z plane.

     

8. The number of chunks can be determined by the N spinbox. Turn the spinbox from to .

 

9. Now there are 8 copies of the planet gear moiety present.

10. Selecting a single white sphere will line up the copies with a certain open bonds attached (depending on which white sphere you choose).

 

  • There are several white spheres present in Extrude mode. These are the “bond offset spheres”, they can be turned on and off through the radio buttons on the dashboard.

 

11. Turn the moiety with the MMB so that the white spheres face you.

 

12. Select the centermost sphere.

 

  • Turning the object with the MMB you can see the where the bonds will form when the done checkmark is selected.

  • Selecting any of the spheres will line up the copies for bonding. The centermost of the "bond offset spheres" is the selection that will bond the most atoms together.
  • Different manipulation of the Bond offset spheres will present differently bonded extrusion.
  • The attached spheres appear as a blended sphere with half of its color green and half blue.
  • Bonds are only formed with the extrusion tool when the green and blue spheres are present.

 

12. Select the done check mark to bond the multi-colored spheres.

  • This will place you back in Select Chunks mode.

 

13. Select the planet gear and press the Hydrogenate icon.

 

14. Select the Save As from the file drop down menu.

 

15. Save the part in the others file folder partlib/mmplib/others/ as planetgear.mmp.
