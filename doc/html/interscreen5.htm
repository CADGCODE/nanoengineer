Home

 

When you create a new model, it appears in a default view orientation (FRONT view).

When you open an existing model, it appears in the orientation it was last saved. You can change the default orientation by selecting Set Home View to Current View from the View menu.

 

Fit to Window

 

 

Refits the model to the screen so you can view the entire model.

 

Zoom Window  
  The zoom window tool enables a closer (zoomed in) view of an area on a chunk.

Ortho

 

 

 

Ortho produces a view of an object that is orthogonal to the view of the user.

Ortho enables a view of a three dimensional part to be viewed at an angle where the part appears to be two dimensional.

 

Perspective

 

 

 

When using the perspective viewing mode a lattice will not line up to be viewed as a two dimensional structure dimensional appearance.
