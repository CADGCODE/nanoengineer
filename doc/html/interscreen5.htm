Home

 

When you create a new model, it appears in a default view orientation (FRONT view).

When you open an existing model, it appears in the orientation it was last saved. You can change the default orientation by selecting Set Home View to Current View from the View menu.

 

Fit to Window

 

 

Refits the model to the screen so you can view the entire model.

 

Recenter

  

 

 

 

Changes the view center and zoom factor so that the origin is in the center of the view and you can view the entire model.
Zoom Window  

 

 

 

Allows the user to zoom into a specific area of the model by specifying a rectangular area.This is done by holding down the left button and dragging the mouse. A mouse with a mouse wheel can also be used to zoom in and out at any time, without using the Zoom Tool.
Pan Tool  

 

 

 

Allows X-Y panning using the left mouse button. Users with a 3-button mouse can pan the model at any time by pressing the middle mouse button while holding down the Shift key.
Rotate Tool  

 

 

 

Allows free rotation using the left mouse button. Users with a 3-button mouse can rotate the model at any time by pressing the middle mouse button and dragging the mouse.

Ortho

 

 

 

Sets nonperspective (or parallel) projection, with no foreshortening.

 

Perspective

 

 

 

Set perspective projection, drawing objects slightly larger that are closer to the viewer.
