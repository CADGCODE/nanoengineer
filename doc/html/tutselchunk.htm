How to select chunks.

1. Hold the RMB and choose to select either atoms or chunks.

  • There are two types of selection in nanoENGINEER-1™. Select Chunks mode and Select Atoms mode.
  • This can also be done by selecting the Select Chunks icon .

2. Pressing the LMB over the desired chunk in Select Chunks mode will select the clicked Chunk.

  • Holding the LMB and dragging a bounding box around a desired section of chunks will select the chunks bounded by the box.
  • A lasso option for selecting chunks appears within the bounding box. If the user draws a circle around the chunks to be selected while holding the LMB this will also select the bounded atoms.
